1. 概要

Markdown はマークアップ言語の一種です。 マークアップとは、HTML の様に「平の文」の見た目を整える為の書き方です。 基本的にはテキスト文書なので、ファイルのサイズが小さくて済みます。

特に Markdown においては、HTML よりも簡単に早く記載できる一方で、HTML と同等の見た目を得ることができます。 使いこなせればグラフもテキストで書くことができるので、非常に便利に使うことができます。

拡張子は .md もしくは、.markdown が使われます。一般的には .md を利用することが多いです。

これは蛇足ですが、このページも Markdown を用いて書いています。

1.1 Markdownを使用するメリット

  • プレビュー表示でなくても見やすい
  • Chrome 等のブラウザで表示できる
  • VSCode を使うと、VSCode が記入のサポートをしてくれて便利
    • 箇条書きの頭を毎回入れる必要が無いなど
  • Markdown の書き方が使えるサービスが多数ある (3. 補足にて記載しています)
  • HTMLへの変換が出来る

2. 導入準備

2.1 VSCode に拡張機能の追加

  1. VSCode を開く
  2. 左側サイドバーより、拡張機能のボタンをクリック
  3. 「Markdown All in One」と検索して、インストールをクリック (画像ではアンインストールとなっています)

2.2 ブラウザに拡張機能の追加

  1. MarkDown Preview Plus をブラウザに追加
  2. 右上の3点リーダーから、拡張機能を管理をクリック
  3. Markdown Preview Plus の詳細をクリック
  4. 「ファイルのURLへのアクセスを許可する」をONにする

3. VSCode を用いた Markdown の書き方

3.1 VSCode の使い方

  1. Ctrl + K を同時に押す (ここでは何も起きません)
  2. Ctrl + O を同時に押す (エクスプローラーが開きます)
  3. Markdown ファイルを作成するフォルダを開き、フォルダの選択をクリック (下記例では、C:\Users\834014\Desktop\研修資料\test\ を開いています。)

3.2.3.1 コードブロック

書き方

` (バッククォート)3つで囲みます。 また、始まりの際は言語を追記します。

```python
def square(num):
   ans = num ** 2
   return ans
```

出力結果

def square(num):
   ans = num ** 2
   return ans
3.2.3.2 インライン

書き方

` (バッククォート)で囲みます。

例えば5の2乗が知りたいときは、`square(5)`と書きます。

出力結果

例えば5の2乗が知りたいときは、square(5)と書きます。
