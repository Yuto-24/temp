1. 概要

Markdown はマークアップ言語の一種です。
マークアップとは、HTML の様に「平の文」の見た目を整える為の書き方です。
基本的にはテキスト文書なので、ファイルのサイズが小さくて済みます。

特に Markdown においては、HTML よりも簡単に早く記載できる一方で、HTML と同等の見た目を得ることができます。
使いこなせればグラフもテキストで書くことができるので、非常に便利に使うことができます。

拡張子は .md もしくは、.markdown が使われます。一般的には .md を利用することが多いです。

1.1 Markdownを使用するメリット

  • 素の分でも見やすい
  • VSCode を使うと、自動補完があり便利
  • Markdown の書き方が使えるサービスが多数ある (3. 補足にて記載しています)

2. 導入方法

2.0 準備

2.0.1 VSCode に拡張機能の追加

  1. VSCode を開く
  2. 左側サイドバーより、拡張機能のボタンをクリック
  3. 「Markdown All in One」と検索して、インストールをクリック
    (画像ではアンインストールとなっています)

2.0.2 ブラウザに拡張機能の追加

  1. MarkDown Preview Plus をブラウザに追加
  2. 管理画面を開き、ファイルのURLへのアクセスを許可するをONにする

2.1 VSCode を用いた Markdown の書き方

2.1.1 VSCode の使い方

  1. Ctrl + K を同時に押す (ここでは何も起きません)
  2. Ctrl + O を同時に押す (エクスプローラーが開きます)
  3. Markdown ファイルを作成するフォルダを開き、フォルダの選択をクリック
    (下記例では、C:\Users\834014\Desktop\研修資料\test\ を開いています。)
